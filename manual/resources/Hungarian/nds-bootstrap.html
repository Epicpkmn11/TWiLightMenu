Az NDS Bootstrap-ről

Az nds-bootstrap egy homebrew alkalmazás, amit a TWiLight Menu++ használ a cartridge mentések és DS-mód homebrew alkalmazások betöltésére a Nintendo DSi/3DS SD kártyájáról.

Létezik egy verzió a B4DS flash kártyákhoz, ami kizárólag homebrew-ra alkalmas kártya és alacsony kompatibilitású flash kártyákhoz.

Kompatibilitás

Ahhoz, hogy megtekintse mely játékok kompatibilisek az nds-bootstrap-pel, tekintse meg a kompatibilitási listát:
cutt.ly/nds-compatibility

A legjobb B4DS kompatibilitáshoz javasoljuk a DS Memory Expansion Pak beszerzését.

Cheats
nds-bootstrap can apply Action Replay cheats via a usrcheat.dat database, which would need to be placed at sd:/_nds/TWiLightMenu/extras. Once the database is placed there, enter your game's per-game setting menu and open the cheats menu using the button prompt shown.
Need to find a database? We recommend using the one from DeadSkullzJr: cutt.ly/DSJCheats.
Alternatively, if you would like to make one yourself, feel free to use the R4CEE application on a Windows device (or Wine if you use Linux or macOS).
