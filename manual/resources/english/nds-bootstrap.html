About

nds-bootstrap is a Nintendo DSi Homebrew used by TWiLight Menu++ to launch Cartridge dumps and Nintendo DS-mode homebrew binaries from the Nintendo DSi/Nintendo 3DS SD card.

There is also a version for Nintendo DS-mode Flashcarts titled B4DS, intended for homebrew-only flashcarts and launching via external nds launchers.

Compatibility

To see if a game is compatible with nds-bootstrap, check the compatibility list: https://bit.ly/nds-compatibility

For the best compatibility with B4DS, we recommend getting a DS Memory Expansion Pak.
