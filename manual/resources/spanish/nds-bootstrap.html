Acerca de

nds-bootstrap es un Homebrew de Nintendo DSi usado por TWiLight Menu++ para lanzar dumps de cartucho y binarios homebrew del modo Nintendo DSi/Nintendo 3DS SD.

También hay una versión para Nintendo DS Flashcarts titulado B4DS, destinada a flashcarts homebrew-only y lanzados a través de lanzadores de nds externos.

Compatibilidad

Para ver si un juego es compatible con nds-bootstrap, revisa la lista de compatibilidad: https://bit.ly/nds-compatibility

For the best compatibility with B4DS, we recommend getting a DS Memory Expansion Pak.
