1я Страница -- настройка GUI
Вы можете выбрать тему для просмотра и другие параметры, влияющие на меню выбора игр.
2я Страница -- настройка эмулятора
Вы можете выбрать, какой эмулятор использовать и для какой платформы в селекторе игр.
3я Страница -- настройка GBARunner2
Это параметры для Gameboy Advance Hypervisor, GBARunner2. Переключить параметры маски, границы или темные цвета.
4я Страница -- настройка игр и приложений
Это настройки запуска приложений. Выберите, использовать ли флешкартриджи или nds-bootstrap, LED индикаторы в качестве чтения (только DSi), широкоэкранный режим только для (3DS) и многое другое.
5я Страница -- другие. Настройки
Опции этой страницы переключают язык для использования, автоматическая загрузка последней воспроизведенной игры, показывать ли всплывающие экраны при запуске и другое.
